# EyeScan
EyeScan is scan tools you can check particular port with host or Ipaddress


# Installation
1- sudo apt-get update

2- sudo apt-get install python3-pip


# Usage 
git clone https://github.com/mohammadfaidi/EyeScan.git

chmod +x EyeScan.py
